## 四、微信支付绑定（任务 3）

**位置**：mp.weixin.qq.com → 微信支付 → 关联商户号

| 项 | 值 |
|---|---|
| 商户号 mchid | ⚠️ **以你 pay.weixin.qq.com 后台显示的 17 位 / 10 位为准**（你给我的两个号有歧义，必须现场确认） |
| 微信支付公钥 ID | `PUB_KEY_ID_0117455102922026051200211756004802`（后端已配） |
| APIv3 密钥 / 商户证书 | 已在服务器 `/www/shangcheng/secrets/wxpay/` 配齐（无需后台操作） |

**操作步骤**：
1. 在 mp.weixin.qq.com 点「关联商户号」→ 填 mchid → 提交
2. 商户号的**管理员手机**会收到验证短信
3. 1 分钟内确认 → 关联成功
4. 关联后 `wx.requestPayment` 才能在小程序里真的弹起付款窗

## 五、订阅消息模板（任务 4，可选但建议）

**位置**：mp.weixin.qq.com → 功能 → 订阅消息 → 我的模板 → 从公共模板库添加

### A. 订单发货通知

- 搜索关键词：**「发货」**，选其中字段最接近下列的一个：
  - `character_string1` 订单编号
  - `character_string2` 物流单号
  - `thing3` 商品名称
- 申请通过后拿到 **tmplId**（形如 `_pCXmkF_qhDxxxxxxx`）

### B. 售后处理结果

- 搜索关键词：**「售后」** 或 **「处理结果」**，选字段最接近：
  - `character_string1` 订单编号
  - `phrase2` 处理结果（如「同意退款」「补发完成」）
  - `thing3` 备注

**拿到两个 tmplId 后告诉我**，我会写到后端 `.env`：
```
WX_TMPL_SHIPPED=_pCXxxx...
WX_TMPL_AFTERSALE=_pCYxxx...
```
并在 `OrderService.ship()` 和 `AftersaleService.resolve()` 里调发送逻辑。

## 六、上传发布流程（任务 5）

```
1. 本地或服务器 git pull  → 拉到最新 main（commit 4cbb693 之后）
2. pnpm install            → 装齐 socket.io-client 等新依赖
3. pnpm --filter @jiujiu/user-mp build:mp-weixin
   产物：packages/user-mp/dist/build/mp-weixin/
4. 提审前自检：pnpm tsx scripts/preflight-mp.ts
   (检查域名、API、wss、证书过期)
5. 打开微信开发者工具 → 导入项目 → 选 dist/build/mp-weixin 目录
   AppID 已自动填 wxe8ed8b7d9d154165
6. 真机预览扫码 → 按【八、真机自测清单】跑一遍
7. 自测全过 → 工具右上角「上传」→ 填版本 1.0.0 + 备注
8. 回 mp.weixin.qq.com → 版本管理 → 找「开发版本」→ 提交审核
9. 审核 1-3 天通过 → 点「发布」上线
```

## 十、常见错误对照表

| 现象 | 大概率原因 | 排查 |
|---|---|---|
| `request:fail url not in domain list` | request 合法域名漏 | 任务 1 重核 |
| 商品图片显示不出来 / 详情图全是占位 | downloadFile 漏 | 任务 1 重核 |
| 聊天页一直 connecting | socket 合法域名漏 / wss 未配置 | preflight 脚本第 6 项 |
| 支付不弹窗，console 报 `requestPayment:fail` | mchid 没绑 / 公钥 ID 错 | 任务 3 + 检查 secrets 目录 |
| 进首页白屏 / `ReferenceError: xxx is not defined` | 服务器 dist 没拉最新 commit | 服务器 git pull && rebuild |
| 地图选址没结果 | 腾讯地图 Key 域名白名单漏 | 腾讯位置服务后台 → Key → 配额 → 域名白名单 |
| 审核驳回「类目不符」 | 服务类目和实际业务不一致 | 重新选类目 → 需要资质（电子商务平台经营备案） |

## 十一、提审后

- 审核时间：**1-3 个工作日**（电商类目偶尔会到 5 天）
- 审核期间用户可访问的版本：**线上版本**（你这次是首版，所以**审核期间用户能看到的只有"开发版"或"体验版"**——首次审核通过点「发布」之后才算真正上线）
- 审核驳回：去 mp.weixin.qq.com → 版本管理 看驳回原因，修完重新提
