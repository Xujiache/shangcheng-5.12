## 一句话

为商城 5.0 新增一个**与商城零耦合的独立产品**——「门窗利账」原生微信小程序记账端，配套后端 `ledger` 域与 admin-pc 后台会员管理，实现「你在后台建账号 / 加会员时长 ↔ 店主小程序登录 / 过期闸门」完整闭环。

## 关键设计落点（对照 4 项决策）

- **D1 技术栈**：原生微信小程序（TS + IDE 内置编译），`packages/ledger-mp` 独立工具链，不与 uni-app 共用。
- **D2 后端域**：全新独立 `Ledger*` 表，与商城零外键耦合；共用 JWT 基础设施，靠 `scope:'ledger'` + 独立 `LedgerJwtGuard` 隔离。
- **D3 会员开通**：仅 admin-pc 后台开通/续费，App 内无支付（续费按钮引导联系管理员）。
- **D4 时长算法**：叠加 `max(now,当前到期)+N 天`，封顶 10 年；后台可选套餐或自定义天数。

## 质量证据

- **三端类型检查全绿**：server `tsc`（src 干净）/ admin-pc `vue-tsc`（exit 0）/ ledger-mp `tsc`（exit 0）。
- **三路多 Agent 对抗验证**：后端安全/隔离（零 P0/P1，IDOR 无法构造、闸门无法绕过、跨域双向拒绝、时长算法正确）、前后端契约（mp↔后端 + admin↔后端 完全一致）、小程序运行时（无死链/空渲染，可在微信工具编译运行）。

## 工程化方式
